# Lox Project

Tree-walking interpreter written in Kotlin and bytecode virtual machine written in Zig for the Lox programming language from [Crafting Interpreters][0].

# Progress

- [x] `lox-kt`
- [ ] `lox-zg`
  - [ ] local-scoped variable
  - [ ] control flow
  - [ ] function and closure
  - [ ] garbage collector
  - [ ] class-based OOP
  - [ ] compiling to WASM
- [ ] `lox-playground`
  - [ ] online playground
